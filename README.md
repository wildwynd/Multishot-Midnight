Multishot Midnight is an event-driven screenshot addon for World of Warcraft: Midnight that automatically captures important moments cleanly, reliably, and without interrupting gameplay.

This project is a modernized fork of the original Multishot addon by Nukme, rebuilt and extended for the Midnight expansion.

---

Features

Automatic screenshots for:
Achievements
Level-ups
Boss and rare kills
Manual triggers

Optional Midnight-style overlay (watermark)
Optional UI hide for clean screenshots
Slash commands and keybind support
Modern Settings UI integration
Safe localization handling (no AceLocale crashes)

/ms wm
