# Roadmap

This roadmap outlines planned improvements for **Multishot Midnight**.
Timelines are flexible and based on stability, not pressure.

---

## Short Term (12.0.14)
- Improve watermark cleanup after screenshots
- Add explicit Delve completion tuning
- Minor Settings UI polish
- Internal code cleanup and documentation

---

## Mid Term (12.1.x)
- Optional per-event watermark toggles
- Better PvP end-of-match detection
- Performance cleanup for large sessions

---

## Long Term
- Expansion transition readiness
- Optional profile support
- Continued maintenance and compatibility updates

---

## Philosophy
Multishot Midnight is intentionally:
- Lightweight
- Stable
- Event-driven
- Easy to understand and configure

Features will only be added if they improve reliability or clarity.
